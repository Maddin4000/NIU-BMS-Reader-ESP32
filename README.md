# NIU-BMS-Reader-ESP32
Read out BMS Cells from Niu Akku 60V 29AH

Ich habe einen BMS auslese Sketch für Niuz Akku geschrieben, inspiriert durch BMS ReadOut unter anderem.

Du benötigtst einen ESP32C3 oder anderen ESP und ein RS485 Konverter zb.: https://amzn.eu/d/dEIrs2X

Verbinde de/re zusammen mit Pin 6 vom ESP in meinem Fall.

RO = RX 
DI = TX
